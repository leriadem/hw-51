# HW -51 Productivity Dashboard

Домашнє завдання: розширення можливостей React-додатків через інтеграцію спеціалізованих бібліотек.

Проєкт демонструє роботу React Toastify та красиві інтерактивні кнопки.

## Опис проекту

Цей проект — невеликий дашборд для демонстрації роботи React Toastify:

Кнопки для показу успіху, помилки та попередження.

Стильні тости з кольорами та анімацією.

Використано React 18 та Vite для швидкого запуску та розробки.

Всі стилі винесені в CSS-файли (App.css, ToastNotification.css), без inline-стилів.

## Технології

React 18 — бібліотека для створення UI

Vite — швидкий збирач і дев-сервер

React Toastify — бібліотека для сповіщень (тостів)

CSS — стилізація компонентів

## Інструкції з встановлення та запуску

Клонувати репозиторій:
git clone <посилання*на*твій_репозиторій>

Перейти в папку проєкту:
cd my-react-app

Встановити залежності:
npm install

Запустити дев-сервер:
npm run dev

Відкрити в браузері:
http://localhost:5173
